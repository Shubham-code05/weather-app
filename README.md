# Weather App

A simple weather application built using HTML, CSS and JavaScript.

## Features
- Search weather by city name
- Shows temperature
- Shows weather condition
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
